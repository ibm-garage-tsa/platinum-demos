<summary>6 - Set up the parameters to install the pipeline</summary>

<br/>

We’re now going to set up pipeline triggers by linking the GitHub repository to OpenShift pipelines. Now, when you commit a change to GitHub, your OpenShift pipeline runs automatically.

<br/>

1. Set the namespace to be the correct namespace. It should be cp4i. On your terminal window, run this command:

<br/>

```
export NAMESPACE=cp4i
```

<br/>

2. Set the GitHub branch to **main** so the script knows which version of the GitHub artifacts to use.

<br/>

```
export BRANCH=main
```

<br/>

3. Set the GitHub repository to use as the URL of your forked repository (made in section 1). You can see it in your browser address bar.<br/>
<br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-integration-dent-deletion/images/prepare-5-3-1.png" width="800" /><br/>
For example, ours is: https://github.com/rosorio/cp4i-deployment-samples.git. Note that rosorio is our github id, yours will be different.

<br/>

```
export FORKED_REPO=https://github.com/<your_id>/cp4i-deployment-samples.git
```

<br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-integration-dent-deletion/images/dent-deletion-1-2-1.png" width="800" /><br/>

<summary>7 - Create the pipeline</summary>

<br/>

Follow these steps to create the pipeline in OpenShift Pipeline page. 

<br/>

1. In the **cp4i-deployment-samples** folder, open the **DrivewayDentDeletion/Operators folder**.

<br/>

```
cd DrivewayDentDeletion/Operators
```

<br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-integration-dent-deletion/images/prepare-6-1.png" width="800" /><br/>

2. Create the pipeline by running this command.

<br/>

```
./cicd-apply-test-apic-pipeline.sh -n $NAMESPACE -r $FORKED_REPO -b $BRANCH
```

<br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-integration-dent-deletion/images/prepare-6-2.png" width="800" /><br/>
