<summary>Introduction</summary>

<br/>

Today I’ll show how the IBM’sdocument managementcapabilities are used to manage and secure content across the organization. We’ll use ahiring use case to showhow organizations leverage document management to simplify and accelerate the hiring process, while facilitating regulatory and corporate compliance. 

<br/>
 
We’ll see how to configure a centralized repository to share and collaborate on content during the hiring process. I’ll use role-based capabilities to configure document access and redact sensitive information. I’ll manage required approvals using built-in workflow capabilities. We’ll also see how video bookmarking and document merging are used to accelerate decision-making.<br/><br/>
 
This demo uses a hiring example, but the same pattern can be applied to numerous scenarios, such as invoicing, claims management, and loan approval. <br/><br/>

Let’s get started!
