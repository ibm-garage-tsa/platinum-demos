<summary>Introduction</summary>

<br/>

Thank you for attending this Predictive Decisioning demonstration.
<br/>
Today I’ll show how IBM Cloud Pak for Business Automation uses machine learning to enhance customer retention offers. You will learn how to integrate Watson Studio Machine Learning with the Cloud Pak to predict business outcomes. These predictions are used by automated decision services to target retention policies, lower customer retention costs, and eliminate many paper-based processes.
<br/>
Business users can quickly get started creating and using predictions to improve their everyday processes.

Let’s get started!
