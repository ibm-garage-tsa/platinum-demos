<details>

<summary>2 - Process Mining server</summary>

<br/>

<InlineNotification> <strong>NOTE</strong>: The VMs should already be started, but they MUST be restarted for the demo to run properly. </InlineNotification>

<br/>

1. Access your Technology Zone environment using the link in the reservation email. You may also use the IBM Technology Zone’s **My Library** → **My Reservations** page.

<br/>

2. Click the **Shutdown VM(s)** button to shut down all three VMs. Wait for the VMs to shut down. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/prep-2.png" width="600" /><br/>

<br/>

3. Click the **Run VM(s)** button to start all three VMs. <br/><br/><InlineNotification> <strong>NOTE</strong>: If the <strong>Run VM(s)</strong> button is not activated after completing the shutdown, you may need to refresh the page. </InlineNotification><br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/prep-3.png" width="600" /><br/>

4. Click to open the IBM Process Mining Server VM. You will need to wait several minutes for the Process Mining Server to start. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/prep-4.png" width="600" />

<br/>

</details>
