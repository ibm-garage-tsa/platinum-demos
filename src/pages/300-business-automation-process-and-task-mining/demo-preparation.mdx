<details>

<summary>2 - Start the Process Mining VMs</summary>

<br/>

1. Access your Technology Zone environment using the link in the reservation email. You may also use the IBM Technology Zone’s **My Library** → **My Reservations** page.

<br/>

2. Click the **Run VM(s)** button to start the VM. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/prep-3.png" width="600" /><br/>

3. Click to open the **IBM Process Mining Server** VM. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/prep-4.png" width="600" /><br/><InlineNotification> <strong>NOTE</strong>: You will need to wait several minutes for the <strong>Process Mining server</strong> to start. </InlineNotification>

<br/>

**[Go to top](#top)**

</details>

<details>

<summary>3 - Unblock the 'maintenance.admin' user </summary>

<br/>

<InlineNotification> <strong>NOTE</strong>: In the current  Technology Zone Process Mining environment, the user ‘maintenance.admin’ is blocked. Please follow these instructions to work around this issue.</InlineNotification>

<br/>

1. In the ‘IBM Process Mining Server’ VM, open a Terminal window. Click **Activities** (1), then **Terminal** (2).<br/><br/>

<img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/activities-terminal.png" width="200" /><br/>

2. Copy and paste or type the following commands into Terminal:<br/><br/>


```
mongo
```

```
use processmining
```

```
db.accounts.update({'username':'maintenance.admin'},{$set:{blocked:false}})
```

```
exit
```

<br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/terminal-screen.png" width="800" />

## **PREPARE TO GIVE THE DEMO**

<details>

<summary>1 - Open the IBM Process Mining workspace</summary>

<br/>

<InlineNotification> <strong>NOTES</strong>: <br/> • If you're already logged in, skip steps 1-4 and start at step 5 below. <br/> • If a <strong>System Not Registered</strong> registration request appears at the top of the screen, click <strong>X</strong> to close it.</InlineNotification>

<br/>

1. Click **Activities**. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/activities.png" width="200" /><br/>

2. Click **Firefox**. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/firefox.png" width="200" /><br/>

3. Click **IBM Process Mining | Sign in** on the bookmark toolbar. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/toolbar.png" width="500" />
<InlineNotification> <strong>NOTE</strong>: If you see a <strong>Warning: Potential Security Risk Ahead</strong> screen, click <strong>Advanced</strong> and then scroll down and click <strong>Accept the Risk and Continue</strong>. </InlineNotification>
<InlineNotification><strong>NOTE</strong>: If you get a <strong>404 error</strong> instead of the login page the first time you open the Process mining workspace, truncate the url and only use: <br/>https://processmining.processmining.demo</InlineNotification> <br/>


4. Enter '**maintenance.admin**' for the **UserID** (1) and '**IBMDem0s!**' for the **Password** (2), and click **Sign in** (3). <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/sign-in.png" width="300" /><br/>

5. From the **Process mining** workspace, click **Account Closure**. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/process-mining.png" width="800" /><br/>

6. The **Account Closure** model is now open. Click the **Datasource** tab. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/account-closure.png" width="800" /><br/>

7. The **Datasource** tab is now open. You're ready to give the demo. <br/><br/><img src="https://raw.githubusercontent.com/ibm-garage-tsa/platinum-demos/master/src/pages/300-business-automation-process-and-task-mining/images/datasource-tab.png" width="800" /><br/>
